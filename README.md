# Installation:

1) Clone this repo

    git clone git://github.com/alexandremcosta/dotvim.git ~/.vim

2) Create symlinks

    ln -s ~/.vim/vimrc ~/.vimrc  
    ln -s ~/.vim/gvimrc ~/.gvimrc

3) Update submodules

    cd ~/.vim  
    git submodule init  
    git submodule update
